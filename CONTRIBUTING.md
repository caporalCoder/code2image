Contributing
============

Thanks for your interest in contributing! A few quick guidelines:

- Use Node 20+ and Vite for local development
- Install deps with `npm i` and run `npm run dev`
- Keep PRs focused and small when possible
- Follow the existing code style and file structure

Reporting Issues
----------------

- Search existing issues before opening a new one
- Include reproduction steps, environment, and screenshots when helpful

Submitting Changes
------------------

1. Fork and create a feature branch
2. Make your changes and ensure the app runs locally
3. Open a pull request with a clear description and context
